百度位置服务提供了免费的地点检索API，可以根据关键词检索某一行政区、矩形区域、圆形区域内的所有地点，单日2000次免费请求（申请成为开发者之后可提升至5W）。不过百度限制了最多返回20X20条数据，要突破这个限制，可以将一个大的矩形区域划分成多个小矩形区域，然后依次通过小矩形区域的坐标来请求百度API，最后汇总。

1.API简介

  • 官方文档：https://lbsyun.baidu.com/index.php?title=webapi/guide/webservice-placeapi
  • 这个API使用起来十分简单，只需要申请一个API密钥（AK）即可使用，一个简单的GET请求即可获取我们想要的信息：

http://api.map.baidu.com/place/v2/search?query=<检索关键词>&bounds=20.06781,110.311802,40.324317,130.656752&output=json&ak=<你的AK>&page_size=20&page_num=0
query: 检索关键词
bounds: 矩形区域坐标，依次为 左下角纬度、左下角经度、右上角纬度、右上角经度
output: 返回数据格式
ak: 申请的API密钥
page_size: 每页数据，最大20
page_num: 页码，最大20

  • 返回数据格式

2.基本思路

由于百度API返回数据最多400条的限制，我们需要将一个大的区域分割成多个小区域，再通过小区域的坐标去请求API。通过左下角、右上角2个坐标点，即可确定一个矩形区域。

l_lat: 左下角纬度
l_long: 左下角经度
r_lat: 右上角纬度
r_long: 右上角经度

3.代码实现

有了思路之后接下来就是代码实现了，下面我们在地图上选取上海地区的一个矩形区域，分成16 X 16个小矩形区域，请求API，获取小区信息，并写入xls文件中

import xlwt
import xlrd
import numpy as np
import requests


def write_to_excel(wb, sheet, titles, data, file_path, row):
    """将信息写入文件"""
    for title in titles:
        sheet.write(row, title["index"], data[title["key"]])
    wb.save(file_path)


def get_position(l_lat, l_long, r_lat, r_long, ak, keyword, page_num=0):
    """请求百度API"""
    url = "http://api.map.baidu.com/place/v2/search"
    params = {
        "query": keyword,
        "bounds": f"{l_lat},{l_long},{r_lat},{r_long}",
        "output": "json",
        "ak": ak,
        "page_size": 20,
        "page_num": page_num
    }
    response = requests.get(url=url, params=params)
    print(response.json())
    return response.json()


def get_area(l_lat, l_long, r_lat, r_long, step):
    """将一个大区域划分为多个小区域"""
    lat_list = [round(i, 6) for i in np.arange(l_lat, r_lat, (r_lat - l_lat)/step)]
    long_list = [round(j, 6) for j in np.arange(l_long, r_long, (r_long - l_long)/step)]
    print(lat_list)
    print(long_list)
    area_list = []
    for long_index, long in enumerate(long_list[:-1]):
        for lat_index, lat in enumerate(lat_list[:-1]):
            area = (lat, long, lat_list[lat_index + 1], long_list[long_index + 1])
            print(area)
            area_list.append(area)
    return area_list


def get_community(*init_area, titles, file_path, sheet_name, ak, keyword):
    """获取某一个区域内所有地点信息"""
    # 给定初始区域坐标，这里划分为16 X 16个矩形
    area_list = get_area(*init_area, 16)
    data_list = []
    wb = xlwt.Workbook(encoding='utf-8')
    sheet = wb.add_sheet(f"{sheet_name}")
    for title in titles:
        sheet.write(0, title["index"], title["name"])
    row = 1
    for area in area_list:
        resp = get_position(
            *area,
            ak=ak,
            keyword=keyword
        )
        total = resp["total"]
        last_page = int(total / 20)
        for i in range(last_page+1):
            response = get_position(
                *area,
                ak=ak,
                keyword=keyword,
                page_num=i
            )
            for info in response["results"]:
                data = {
                    "name": info["name"],
                    "lat": info["location"]["lat"],
                    "lng": info["location"]["lng"],
                    "address": info["address"],
                    "telephone": info.get("telephone"),
                }
                data_list.append(data)
                write_to_excel(wb, sheet, titles, data, file_path, row)
                row += 1
    return data_list

    ]

        *init_area,
